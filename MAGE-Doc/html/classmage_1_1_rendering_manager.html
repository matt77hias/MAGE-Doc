§ SwitchMode()

void mage::RenderingManager::SwitchMode ( bool  toggle)

Recreates the swap chain buffers and switches the mode of this rendering manager. Windowed mode is switched to full screen mode and vice versa.

Parameters
[in]toggleIf true only the swap chain buffers will be recreated to match the current mode of the swap chain and no mode switch will occurs. If false both the swap chain buffers will be replaced and a mode switch will occur.
